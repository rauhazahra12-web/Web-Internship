# 🌟 Week 1 – Day 3: Premium Personal Portfolio

## 📌 Overview

This project was developed as part of my **Web Development Internship – Week 1, Day 3**. The goal was to design and develop a modern, responsive, and interactive personal portfolio website using HTML, CSS, and JavaScript while focusing on clean UI, smooth animations, and an enhanced user experience.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern glassmorphism-inspired interface
* Dark and light theme support
* Smooth scrolling navigation
* Animated typing effect
* Interactive project showcase
* Skills section with animated counters
* Contact form
* Email copy functionality
* Back-to-top button
* Scroll progress indicator
* Custom cursor spotlight effect
* Clean and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Responsive Web Design
* CSS Animations
* Glassmorphism UI Design
